## 2. Тематика сайта, название
**Название:** Rice World  
**Тема:** Всё о рисе — сорта, рецепты, культура потребления и полезные инструменты для приготовления.  
Сайт объединяет справочную информацию, калькулятор варки, коллекцию рецептов из внешнего API и личный дневник питания.

## 3. Цели и задачи сайта
**Цель:** Создать уютный и полезный ресурс для любителей риса, который помогает разобраться в многообразии сортов, правильно готовить рис и находить новые рецепты.

**Задачи:**
- **Информирование** — страница «Сорта риса» с карточками описывает свойства, вкусовые нюансы и применение каждого сорта (басмати, жасмин, арборио и др.).  
- **Помощь в приготовлении** — интерактивный калькулятор рассчитывает точный объём воды и время варки в зависимости от сорта и количества порций, исключая ошибки пользователя.  
- **Кулинарное вдохновение** — раздел с рецептами загружает из API TheMealDB блюда, в которых используется рис, и визуализирует их в виде карточек.  
- **Отслеживание личного рациона** — дневник питания (LocalStorage) позволяет добавлять записи с описанием блюд и датой, а также удалять их. Связь между элементами контента и целями: карточки сортов → информирование; форма калькулятора → точный расчёт; галерея рецептов → вдохновение; форма дневника → контроль рациона.

## 4. Целевая аудитория
- **Типичный пользователь:** люди от 18 до 45 лет, которые регулярно готовят дома, интересуются кулинарией и хотят улучшить качество блюд из риса.  
- **Потребности в информации:** точные пропорции воды и времени варки для разных сортов, новые рецепты, справочные данные о сортах, возможность вести пищевой дневник.  
- **Особенности аудитории:** ценят наглядность, минималистичный дизайн, быстрый доступ к инструментам без регистрации. Для привлечения внимания использованы тёплые природные цвета, адаптивная вёрстка и две темы (светлая / тёмная).

## 5. Содержание и структура сайта
Сайт состоит из 6 страниц, связанных единым меню (на мобильных — бургер-меню).
**Описание страниц:**
1. **Главная** — баннер с названием, краткая миссия, превью популярных сортов, кнопка переключения темы.  
2. **Сорта риса** — статический контент: карточки с фото и описанием каждого сорта (басмати, жасмин, краснодарский круглозёрный, арборио для ризотто и т.д.).  
3. **Рецепты** — кнопка «Загрузить рецепты», асинхронный запрос к TheMealDB API, отрисовка карточек с изображением и названием блюда, индикатор загрузки.  
4. **Калькулятор** — форма с выбором сорта риса и количества порций; после отправки выводит объём воды, время варки и отстоя.  
5. **Дневник** — форма добавления записи (название блюда, дата, описание), сохранение в LocalStorage, динамический вывод карточек с возможностью удаления.  
6. **О проекте** — информация об авторе и контактная форма (без серверной обработки, демонстрационная вёрстка).
